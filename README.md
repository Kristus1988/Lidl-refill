# 📱 Display Reader App

![Dark Mode](https://img.shields.io/badge/Dark%20Mode-Only-121212?style=for-the-badge&logo=darkreader)
![Android](https://img.shields.io/badge/Android-24%2B-3DDC84?style=for-the-badge&logo=android)
![Kotlin](https://img.shields.io/badge/Kotlin-1.9.20-7F52FF?style=for-the-badge&logo=kotlin)

Eine Android-App zum Auslesen von Display-Bereichen mit **Dark Mode only** und Overlay-Funktionen.

## ✨ Features

- 🌙 **Dark Mode only** - Perfekt für die Nacht
- 🔵 **Kreis-Modus** - Wähle einen kreisförmigen Bereich
- 🟩 **Rechteck-Modus** - Wähle einen rechteckigen Bereich
- 🎯 **Klick-Simulation** - Simuliert Klicks auf ausgewählte Bereiche
- 📊 **Pixel-Analyse** - Zeigt Farbinformationen des Bereichs
- 💾 **Daten-Export** - Speichert Analysen als Textdatei
- 🔄 **Refresh-Funktion** - Setzt alles zurück
