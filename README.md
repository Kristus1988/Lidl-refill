# 🔄 Lidl Refill Automatik

Eine Android-App zur automatischen Refill-Überwachung für Lidl Connect mit Dark Mode und menschlichem Verhalten.

## ✨ Features

- 🌙 **Dark Mode only**
- 🔵 **Refill Button markieren** - Kreis auf Refill Button ziehen
- 🟩 **Volumen-Anzeige markieren** - Rechteck auf Volumen ziehen
- 🧠 **Menschliches Verhalten** - Zufällige Verzögerungen, Abweichungen, natürliche Bewegungen
- 🔔 **Benachrichtigungen** - Bei automatischen Aktionen
- 📊 **Volumen-Überwachung** - Automatische Prüfung
- 📋 **Aktivitäten-Log** - Alle Aktionen werden protokolliert

## 📱 Bedienung

1. **Lidl Connect App öffnen**
2. **"Refill Button markieren"** - Kreis auf Refill Button ziehen
3. **"Volumen-Anzeige markieren"** - Rechteck auf Volumen ziehen
4. **"Automatisierung STARTEN"** - Startet die Überwachung
5. **Automatischer Refill** - Bei Volumen unter 0.35 GB

## 🔒 Berechtigungen

- **SYSTEM_ALERT_WINDOW** - Overlay anzeigen
- **FOREGROUND_SERVICE** - Service im Hintergrund
- **POST_NOTIFICATIONS** - Benachrichtigungen
- **BIND_ACCESSIBILITY_SERVICE** - Für Klicks und Swipes
